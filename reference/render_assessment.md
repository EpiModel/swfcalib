# Generate an html report of the auto-calibration

The report contains descriptions of the parameters spaces and residual
errors over the duration of the calibration.

## Usage

``` r
render_assessment(
  path_to_assessments,
  output_filename = "assessment.html",
  output_dir = NULL
)
```

## Arguments

- path_to_assessments:

  Path to an `assessments.rds` file generated by an `swfcalib` process.

- output_filename:

  Name of the html report (default = "assessment.html")

- output_dir:

  Directory where to store the report (default = current working
  directory)
